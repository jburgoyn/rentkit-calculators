# Event Timeline Planner

## Overview

An interactive tool that helps event hosts create detailed day-of timelines, including rental delivery/pickup windows, setup times, and vendor coordination schedules.

## Problem Statement

Event timing coordination is complex:
- Rental delivery must happen before setup
- Multiple vendors need access at different times
- Venue access windows are limited
- Setup time is often underestimated
- Day-of chaos when timeline isn't clear

## Target Audience

- **Primary:** DIY event hosts without coordinators
- **Secondary:** Venues helping clients plan
- **Tertiary:** Event coordinators streamlining planning

## SEO Opportunity

- "wedding day timeline" - 5,400 monthly searches
- "event timeline template" - 2,900 monthly searches
- "wedding schedule planner" - 1,600 monthly searches
- "party timeline maker" - 800 monthly searches

## Key Features

### 1. Event Type Templates
- Wedding ceremony + reception
- Reception only
- Corporate event
- Birthday party
- Graduation party
- Custom

### 2. Key Time Inputs
- Event start time
- Venue access time
- Ceremony time (if applicable)
- Reception/party start
- Event end time
- Venue must-vacate time

### 3. Auto-Generated Timeline
System automatically calculates:
- Rental delivery window
- Setup time needed (based on items rented)
- Vendor arrival times
- Getting ready times
- Buffer periods
- Breakdown time
- Rental pickup window

### 4. Customization
- Add custom events
- Adjust durations
- Add notes
- Assign to vendors/people
- Mark as flexible vs fixed

### 5. Vendor Timeline Views
- Generate vendor-specific timelines
- Rental company sees delivery/pickup only
- Caterer sees their relevant times
- Photographer sees photo moments

### 6. Export & Sharing
- PDF download
- Shareable link
- Email to vendors
- Print-friendly version
- Calendar integration (ICS)

## Technical Requirements

### Data Model

```typescript
interface EventTimeline {
  id: string;
  eventName: string;
  eventDate: Date;
  eventType: EventType;

  keyTimes: {
    venueAccess: string;        // "10:00 AM"
    ceremonyStart?: string;
    receptionStart: string;
    eventEnd: string;
    venueMustVacate: string;
  };

  events: TimelineEvent[];

  vendors: Vendor[];
  notes: string;
}

interface TimelineEvent {
  id: string;
  title: string;
  startTime: string;
  endTime: string;
  duration: number;           // minutes
  category: EventCategory;
  assignedTo?: string[];
  location?: string;
  notes?: string;
  isFixed: boolean;           // Can't be moved
  isAutoGenerated: boolean;
}

type EventCategory =
  | 'setup' | 'rental' | 'vendor' | 'getting_ready'
  | 'ceremony' | 'photos' | 'reception' | 'breakdown';

interface Vendor {
  name: string;
  type: VendorType;
  contact?: string;
  arrivalTime?: string;
  departureTime?: string;
}
```

### Timeline Generation Logic

```typescript
const SETUP_TIME_ESTIMATES = {
  tables: 2,              // minutes per table
  chairs: 0.5,            // minutes per chair
  linens: 1,              // minutes per table (including dressing)
  centerpieces: 3,        // minutes per table
  danceFloor: 30,         // base minutes + per panel
  tent: 120,              // 2-4 hours depending on size
  lighting: 45,           // basic string lights
  audioVisual: 60,        // DJ setup
};

function generateTimeline(
  keyTimes: KeyTimes,
  rentalItems: RentalItem[],
  eventType: EventType
): TimelineEvent[] {
  const events: TimelineEvent[] = [];

  // Calculate total setup time
  const setupMinutes = calculateSetupTime(rentalItems);

  // Work backwards from ceremony/event start
  const eventStart = parseTime(keyTimes.ceremonyStart || keyTimes.receptionStart);

  // Rental delivery (2 hours before setup starts)
  const setupStart = subtractMinutes(eventStart, setupMinutes + 30); // 30 min buffer
  const deliveryWindow = {
    start: subtractMinutes(setupStart, 60),
    end: setupStart
  };

  events.push({
    id: 'rental-delivery',
    title: 'Rental Delivery Window',
    startTime: formatTime(deliveryWindow.start),
    endTime: formatTime(deliveryWindow.end),
    duration: 60,
    category: 'rental',
    isFixed: false,
    isAutoGenerated: true,
    notes: 'Tables, chairs, and linens delivered'
  });

  events.push({
    id: 'venue-setup',
    title: 'Venue Setup',
    startTime: formatTime(setupStart),
    endTime: formatTime(subtractMinutes(eventStart, 30)),
    duration: setupMinutes,
    category: 'setup',
    isFixed: false,
    isAutoGenerated: true
  });

  // Add standard events based on event type
  if (eventType === 'wedding') {
    events.push(...generateWeddingEvents(keyTimes));
  }

  // Breakdown and pickup
  const eventEnd = parseTime(keyTimes.eventEnd);
  events.push({
    id: 'breakdown',
    title: 'Breakdown & Cleanup',
    startTime: formatTime(eventEnd),
    endTime: formatTime(addMinutes(eventEnd, 60)),
    duration: 60,
    category: 'breakdown',
    isFixed: false,
    isAutoGenerated: true
  });

  events.push({
    id: 'rental-pickup',
    title: 'Rental Pickup Window',
    startTime: formatTime(addMinutes(eventEnd, 60)),
    endTime: formatTime(parseTime(keyTimes.venueMustVacate)),
    duration: 60,
    category: 'rental',
    isFixed: false,
    isAutoGenerated: true,
    notes: 'All rental items staged for pickup'
  });

  return events.sort((a, b) =>
    parseTime(a.startTime).getTime() - parseTime(b.startTime).getTime()
  );
}
```
